# 项目功能
```
1.前端支持界面：主页、登录、注册、数据展示
2.后端支持
    - 登录、注册、密码重置
    - 创建数据库模型，存储用户数据和应用数据（基于json文件存储）
    - 实现基本的CRUD操作
        - 新建：对应注册功能
        - 读取：注册和登录的校验均有应用
        - 更新：对应 保存更改 功能
        - 删除：对应 注销账户 功能
```
# 使用方法
```
cd /path/to/selfwork
# 后端启动
nohup node ./backend/server.js &
# 前端启动
cd frontend
npm run serve
```
